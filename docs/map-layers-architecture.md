# Архитектура слоёв карты GoApsny

В рамках обновления GoApsny, мы отходим от простой визуализации доступности и превращаем карту в многослойную платформу. 

## Логическая структура слоёв

### 1. Слой «Светофорная доступность» (Базовый)
- **Суть:** Классический Wheelmap-слой: Красный/Желтый/Зелёный/Серый.
- **Источник данных:** REST API wheelmap.org (теги `wheelchair=yes/no/limited`).
- **Состояние слоя:** Включен по умолчанию.

### 2. Слой «Пандус по запросу»
- **Суть:** Подсветка мест, оборудованных мобильными/приставными пандусами (программа-аналог *StopGap*).
- **Иконка:** `/public/icons/portable-ramp.svg` (Оранжевый маркер с коляской внутри).
- **Источник данных (Архитектурное решение):** Поскольку тег переносных пандусов слабо распространен в мировом OSM, рекомендуется вести **отдельный внутренний GeoJSON-файл / базу Supabase** или договориться о простановке тега `wheelchair:portable_ramp=yes` в OSM. 
- **Z-Index:** Этот слой должен рендериться *поверх* базового светофорного слоя. Если объект доступен только за счет мобильного пандуса, его базовая доступность в OSM часто = `limited`. Оранжевая иконка перекроет желтую точку на карте, выделяя место.

### 3. Слой «Городская навигация» 
- **Суть:** Объекты с пилотными навигационными элементами (указатели входов).
- **Источник данных:** На старте — это статический GeoJSON, зашитый в `src/constants/navigation-mock.json`.
- **Объекты (Пилот):**
  1. **Аптека** — *Входной указатель*
  2. **Филармония** — *Указатель доступного альтернативного входа*
  3. **Транспортный узел** — *Транспортная навигация*
- **Визуализация:** Специальные маркеры "щита" или "путеводителя", привязанные к конкретным координатам. Кликом по маркеру открывается карточка-фотография навигационного элемента (как доказательство изменений).

## Интеграция в Next.js (Wheelmap)

В компоненте карты (например `MapInterface`) нужно создать State:
```typescript
const [activeLayers, setActiveLayers] = useState({
  base: true,
  ramps: false,
  navigation: false
});
```

Для элементов управления в UI (помимо фильтров категорий) добавляется панель **"Слои"**. При переключении тумблеров, будут добавляться или скрываться соответствующие Mapbox `Source` и `Layer` компоненты.
